# Honeypot-Monitoring-Platform-using-Cowrie-and-Wazuh

Threat Intelligence-Enriched Honeypot Monitoring Platform
Overview

This project implements a security monitoring platform that integrates a Cowrie SSH honeypot with Wazuh SIEM to detect and analyze malicious activities targeting Linux systems. The solution enriches detected events using AbuseIPDB and VirusTotal and sends real-time notifications through Discord webhooks.

Key Features :
  Cowrie SSH honeypot deployment
  Wazuh SIEM integration
  Custom Wazuh detection rules
  Threat intelligence enrichment using AbuseIPDB and VirusTotal
  Dynamic risk classification
  Real-time Discord notifications
  Hydra-based attack simulation
Learning Outcomes :
  Honeypot deployment and management
  SIEM log analysis and event correlation
  Custom Wazuh rule development
  Threat intelligence integration
  Security automation using Python
  Real-time SOC alerting
